# 车位置换业务审核 Skill

## 描述

对车位置换业务进行逐项审核。将上传的车场资料数据与 BEM 系统数据进行对比分析，按检查项输出初审结果。

## 触发条件

- 资料完整性检查通过 + 业务类型为"车位置换业务"时触发

## 前置依赖

- `completeness-check` 已通过
- `bem-fetch` 已获取 BEM 数据（临停收入、月票收入、月票类型）
- `company-lookup` 已获取企查查数据（参保人数）

## 输入

| 数据 | 来源 |
|------|------|
| 上传的车场资料 | 运营上传，由 Harness 解析 |
| BEM 临停收入数据 | bem-fetch skill |
| BEM 月票收入数据 | bem-fetch skill |
| BEM 月票类型数据 | bem-fetch skill |
| 企查查参保人数 | company-lookup skill |

## 检查项清单

### 1. 临停收入对比

| 对比项 | 资料来源 | BEM来源 | 判定 |
|--------|---------|---------|------|
| 月均临停收入 | 测算表-车场月均临停收入 | 临停月均(实收金额-微信) | 一致✅ / 不一致⚠️输出差异数据 |

若 BEM 数据有下滑趋势，额外输出趋势说明。

### 2. 月票收入对比

| 对比项 | 资料来源 | BEM来源 | 判定 |
|--------|---------|---------|------|
| 月均月票收入 | 测算表-车场月租收入 | 月票月均收入 | 一致✅ / 不一致⚠️输出差异数据 |

### 3. 合同有效年限检查

- **规则**：合同有效年限（月）必须在承包期限内
- **附加规则**：合同有效年限必须 ≥ 3年（36个月）
- **判定**：范围内且≥3年✅ / 超出承包期❌ / 不足3年⚠️

### 4. 对外月卡费用对比

- **规则**：测算表中的月租价格应与 BEM 系统中"对外办理"的月票套餐价格一致
- **数据来源**：BEM 月票类型中 `is_internal = false` 的套餐
- **判定**：一致✅ / 不一致⚠️输出差异

### 5. 设备服务置换金额检查

- **规则**：设备服务置换金额若与 CRM 结算价一致，需与业务确认是否填错
- **说明**：设备服务置换金额应为 CRM 的报价单总金额，而非 CRM 结算价
- **判定**：不一致✅ / 一致📋标注"需与业务确认是否填错"

### 6. 是否允许张贴物料

- **规则**：不允许时需标注
- **判定**：允许✅ / 不允许📋标注"需业务二次明确沟通"

### 7. 毛利率计算

```
毛利率 = 项目整体毛利 / 停车券总价值
```

- 输出毛利率百分比
- 结合实际采买折扣输出对应业务审核权限

### 8. 合作客户主体 — 企查查

- 输出参保人数
- 承包方时额外检查：承包到期时间是否超出合同时间

### 9. 信息完整度

- 所有该填的都得填（completeness-check 的二次确认）

## 输出格式

按 `prompts/output_template.md` 统一格式输出，逐项给出 ✅/⚠️/❌/📋 标记。
