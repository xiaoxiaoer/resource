# 停车券业务审核检查清单

以下为停车券业务的完整检查项，审核时按此清单逐项执行。

## 检查清单

### 第一阶段：资料完整性

| 序号 | 检查项 | 必填条件 |
|------|--------|---------|
| 1 | 合作客户主体 | 始终必填 |
| 2 | 是否产权方/承包方 | 始终必填 |
| 3 | 承包到期时间 | 承包方时必填 |
| 4 | 合同有效年限（月） | 始终必填 |
| 5 | 是否存在自有小程序/ETC | 始终必填 |
| 6 | 车场名称 | 始终必填 |
| 7 | 车场月均临停收入 | 始终必填 |
| 8 | 车场月均月票收入 | 涉及月票置换时必填 |
| 9 | 券消耗年限（月） | 始终必填 |
| 10 | 停车券月均消耗额度 | 始终必填 |
| 11 | 停车券总价值 | 始终必填 |
| 12 | 项目整体毛利 | 始终必填 |

### 第二阶段：数据对比与风险检查

| 序号 | 检查项 | 对比数据 | 判定规则 |
|------|--------|---------|---------|
| C1 | 临停收入 | 测算表 vs BEM实收金额-微信 | 一致✅ / 不一致⚠️ |
| C2 | 月票收入 | 测算表 vs BEM月票月均 | 一致✅ / 不一致⚠️ |
| C3 | 合同有效年限 | 资料中的承包期限 | 在期限内✅ / 超出❌ |
| C4 | 券消耗年限 | vs 合同有效年限 | 晚于✅ / 不晚于❌ |
| C5 | 可消耗收入占比 | BEM渠道明细计算 | 仅自有渠道时触发 |
| C6 | 停车券月均消耗 | ≤ 可消耗占比×月均临停 | 满足✅ / 不满足⚠️ |
| C7 | 分期代扣 | 有/无 | 有则📋人工评估 |
| C8 | CRM专项服务费 | 有/无 | 有则📋人工评估 |
| C9 | 业务费 | 有/无 | 有则📋人工评估 |
| C10 | 毛利率 | 项目整体毛利/停车券总价值 | 输出百分比+权限范围 |
| C11 | 企查查参保人数 | 企查查查询结果 | 输出人数供参考 |
| C12 | 承包到期时间检查 | vs 合同时间 | 承包方时触发，超出❌ |
