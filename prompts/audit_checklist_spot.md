# 车位置换业务审核检查清单

以下为车位置换业务的完整检查项，审核时按此清单逐项执行。

## 检查清单

### 第一阶段：资料完整性

| 序号 | 检查项 | 必填条件 |
|------|--------|---------|
| 1 | 合作客户主体 | 始终必填 |
| 2 | 是否产权方/承包方 | 始终必填 |
| 3 | 承包到期时间 | 承包方时必填 |
| 4 | 合同有效年限（月） | 始终必填，且需≥36个月 |
| 5 | 是否存在自有小程序/ETC | 始终必填 |
| 6 | 车场名称 | 始终必填 |
| 7 | 车场月均临停收入 | 始终必填 |
| 8 | 车场月均月票收入 | 涉及月票时必填 |
| 9 | 设备服务置换金额 | 始终必填 |
| 10 | CRM结算价 | 始终必填 |
| 11 | 对外办理的月卡费用 | 始终必填 |
| 12 | 是否允许张贴物料 | 始终必填 |

### 第二阶段：数据对比与风险检查

| 序号 | 检查项 | 对比数据 | 判定规则 |
|------|--------|---------|---------|
| C1 | 临停收入 | 测算表 vs BEM实收金额-微信 | 一致✅ / 不一致⚠️ |
| C2 | 月票收入 | 测算表 vs BEM月票月均 | 一致✅ / 不一致⚠️ |
| C3 | 合同有效年限 | 资料中的承包期限 | 在期限内且≥36个月✅ / 超出❌ / 不足⚠️ |
| C4 | 对外月卡费用 | 测算表 vs BEM对外月票套餐价 | 一致✅ / 不一致⚠️ |
| C5 | 设备服务置换金额 | vs CRM结算价 | 不一致✅ / 一致📋需确认 |
| C6 | 是否允许张贴物料 | 上传资料 | 允许✅ / 不允许📋需二次沟通 |
| C7 | 毛利率 | 项目整体毛利/停车券总价值 | 输出百分比+权限范围 |
| C8 | 企查查参保人数 | 企查查查询结果 | 输出人数供参考 |
| C9 | 承包到期时间检查 | vs 合同时间 | 承包方时触发，超出❌ |
| C10 | 信息完整度 | 二次确认所有字段 | 全填✅ / 有缺❌ |
